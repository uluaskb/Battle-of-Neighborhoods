# Dog Friendly London Neighbourhoods

This project aims to identify best neighbourhoods for small families with a dog.  Main criterias are:
01    |    Crime rate
02    |    Commute time to Bank station
03    |    2 bedroom rental price 
04    |    Nearby parks
